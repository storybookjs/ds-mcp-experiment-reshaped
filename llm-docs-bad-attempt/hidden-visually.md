# HiddenVisually Component

## Overview
The HiddenVisually component is a utility for hiding content visually while keeping it accessible to screen readers and assistive technologies, essential for inclusive design and accessibility.

## Key Features
- Visual hiding with screen reader accessibility
- SEO-friendly content hiding
- Zero visual footprint
- Maintains semantic structure
- Keyboard navigation support
- Focus management options

## Props Interface
```typescript
type HiddenVisuallyProps = {
  children: React.ReactNode;
  focusable?: boolean;
  className?: string;
  attributes?: React.HTMLAttributes<HTMLSpanElement>;
};
```
