# Text Component

## Overview
The Text component provides consistent text styling, variants, and responsive typography throughout the design system, offering semantic text rendering with theme integration and accessibility support.

## Key Features
- Typography scale integration
- Semantic text variants
- Responsive text sizing
- Color and weight options
- Accessibility compliance
- Theme-aware styling

## Props Interface
```typescript
type TextProps = {
  children: React.ReactNode;
  variant?: 'display-1' | 'display-2' | 'title-1' | 'title-2' | 'title-3' | 'title-4' | 'title-5' | 'title-6' | 'body-1' | 'body-2' | 'body-3' | 'caption';
  weight?: 'regular' | 'medium' | 'semibold' | 'bold';
  color?: 'neutral' | 'neutral-faded' | 'neutral-contrast' | 'primary' | 'positive' | 'critical' | 'warning' | 'disabled';
  align?: 'left' | 'center' | 'right' | 'justify';
  truncate?: boolean | number;
  as?: 'span' | 'p' | 'div' | 'h1' | 'h2' | 'h3' | 'h4' | 'h5' | 'h6';
  className?: string;
  attributes?: React.HTMLAttributes<HTMLElement>;
};
```
