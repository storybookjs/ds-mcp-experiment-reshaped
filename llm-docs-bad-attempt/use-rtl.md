# useRTL Hook

## Overview
The useRTL hook provides access to right-to-left text direction context, enabling components to adapt their layout and behavior for RTL languages like Arabic, Hebrew, and Persian.

## Key Features
- RTL direction detection
- Layout adaptation utilities
- Internationalization support
- Performance optimized
- Theme integration

## Hook Interface

'primary' : 'neutral-faded'}\n                style={{ flex: 1 }}\n              >\n                {stepLabels[step as keyof typeof stepLabels]}\n              </Text>\n            );\n          })}\n        </View>\n        \n        {/* Step Content */}\n        <Card padding={5}>\n          <View gap={3}>\n            <Text variant=\"title-6\">\n              {isRTL ? `الخطوة ${currentStep}` : `Step ${currentStep}`}: {stepLabels[currentStep as keyof typeof stepLabels]}\n            </Text>\n            <Text variant=\"body-3\">\n              {isRTL \n                ? `محتوى الخطوة ${currentStep} سيظهر هنا مع التخطيط المناسب للغة العربية.` \n                : `Content for step ${currentStep} would appear here with appropriate layout.`\n              }\n            </Text>\n          </View>\n        </Card>\n        \n        {/* Navigation Buttons */}\n        <View \n          direction=\"row\" \n          gap={3} \n          justify=\"space-between\"\n        >\n          <Button\n            variant=\"outline\"\n            onClick={() => goToStep(currentStep - 1)}\n            disabled={currentStep === 1}\n          >\n            {buttonLabels.previous}\n          </Button>\n          \n          <Text variant=\"body-3\" color=\"neutral-faded\">\n            {currentStep} / {totalSteps}\n          </Text>\n          \n          <Button\n            onClick={() => {\n              if (currentStep === totalSteps) {\n                alert(isRTL ?
