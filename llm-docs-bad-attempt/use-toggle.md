# useToggle Hook

## Overview
The useToggle hook manages boolean toggle state with convenience methods for toggling, enabling, and disabling states, providing a clean API for binary state management.

## Key Features
- Boolean state management
- Convenient toggle methods
- Explicit enable/disable functions
- TypeScript support
- Performance optimized

## Hook Interface
```typescript
function useToggle(\n  initialValue: boolean = false\n): [\n  boolean,\n  {\n    toggle: () => void;\n    enable: () => void;\n    disable: () => void;\n    set: (value: boolean) => void;\n  }\n];\n```\n\n## Usage Examples\n\n### Basic Toggle Functionality\n```typescript\nimport { useToggle, Button, Text, View, Card } from 'reshaped';\n\nfunction BasicToggle() {\n  const [isEnabled, { toggle, enable, disable, set }] = useToggle(false);\n  \n  return (\n    <Card padding={6}>\n      <View gap={4}>\n        <Text variant=\"title-5\">Basic Toggle Demo</Text>\n        \n        <View \n          padding={4}\n          style={{\n            backgroundColor: isEnabled \n              ?
